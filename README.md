# AI Formula → Word Converter

A lightweight web application that **automatically converts mathematical formulas generated by AI chatbots into Microsoft Word (.docx) format**.

This tool allows users to:

* Paste AI-generated content (ChatGPT, Claude, Gemini, etc.)
* Preview mathematical formulas using KaTeX
* Convert the content into a Word document
* Download a `.docx` file with correctly formatted equations

The system uses **Next.js for the frontend, FastAPI for the backend, and Pandoc for document conversion.**

---

# Demo Workflow

```
AI Chatbot Output
       ↓
Paste into Web App
       ↓
Formula Preview (KaTeX)
       ↓
Send to Backend API
       ↓
Pandoc Conversion
       ↓
Download Word (.docx)
```

# Installation

## 1. Clone the repository

```bash
git clone https://github.com/your-username/ai-formula-word.git
cd ai-formula-word
```

---

# Backend Setup

## 1. Create Python environment

```
cd backend

python -m venv venv
```

Activate environment

Windows

```
venv\Scripts\activate
```

Mac / Linux

```
source venv/bin/activate
```

---

## 2. Install dependencies

```
pip install -r requirements.txt
```

---

## 3. Install Pandoc

Download Pandoc:

[https://pandoc.org/installing.html](https://pandoc.org/installing.html)

Verify installation:

```
pandoc -v
```

Expected output example:

```
pandoc 3.x
```

---

## 4. Run backend server

```
uvicorn app.main:app --reload
```

Backend API will run at:

```
http://localhost:8000
```

API documentation:

```
http://localhost:8000/docs
```

---

# Frontend Setup

## 1. Install dependencies

```
cd frontend

npm install
```

---

## 2. Create environment file

Create:

```

```

## 3. Run frontend

Open:

```

```

# API

## POST /convert

Convert Markdown + LaTeX to Word document.

### Request

```
POST /convert
```

Body

```
{
  "text": "The equation is $$x^2+y^2=z^2$$"
}
```

### Response

```
application/vnd.openxmlformats-officedocument.wordprocessingml.document
```

Returns a downloadable `.docx` file.
